# AI Water Tracker

## Overview

An AI-powered water intake tracking application built with FastAPI, SQLite, and Google Gemini. The system allows users to log their daily water consumption, store hydration records, retrieve intake history, and receive personalized hydration insights generated by an LLM.

## Features

-Log daily water intake through REST APIs
-Store hydration records using SQLite
-Retrieve user-specific intake history
-AI-generated hydration analysis with Google Gemini
-Request validation using Pydantic
-Application logging for monitoring and debugging
-Automatic API documentation with Swagger UI

## Tech Stack
